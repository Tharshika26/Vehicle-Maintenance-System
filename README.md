## Project Description
AutoTrack Web is a web application built with **React** for the frontend and **Python Django** for the backend. It allows vehicle owners to manage their vehicles and track full service history, while service centers (admins) can add official service records. The system provides chronological service history, maintenance cost tracking, service reminders, and reports for both owners and admins.

---

## Features

### Vehicle Owner
- Add, edit, delete vehicles
  - Vehicle number, brand, model, year, fuel type
- View vehicle list
- View full service history in chronological order
- View maintenance cost summary
- Receive next service reminders

### Service Center (Admin)
- Add service records for any vehicle
  - Service date, service type, kilometer reading, cost, notes
- View all service records
- Generate reports and analytics

### Common
- Role-based authentication and protected routes
- Responsive and modern UI
- Dashboard with upcoming service alerts
- Exportable reports (PDF / charts)

---

## Technology Stack
- **Frontend:** React, Tailwind CSS / Bootstrap
- **Backend:** Python Django
- **Database:** PostgreSQL / MySQL (based on setup)
- **API calls:** Axios
- **Version Control:** Git & GitHub
